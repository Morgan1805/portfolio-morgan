# Morgan Portfolio (GitHub Pages)

Clean static portfolio you can host on GitHub Pages.

## What's inside
- `index.html` — main page
- `style.css` — styles
- `script.js` — buttons (Twitter opens, Discord copies username + opens Discord)
- `assets/pfp.jpeg` — your profile image

## Quick setup (GitHub Pages)
1. Create a new repo (example: `morgan-portfolio`)
2. Upload these files/folders to the repo root:
   - `index.html`
   - `style.css`
   - `script.js`
   - `assets/`
3. Go to **Settings → Pages**
4. Under **Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Save — your site link will appear on that same page.

## Edit basics
- Name / headline: open `index.html` and search for `Morgan`
- Twitter link: `https://x.com/Morgan_1805` (in `index.html` + `script.js`)
- Discord username copied: `Morgan_1805` (in `script.js`)

## Notes
Discord doesn’t provide a public username profile URL. The Discord button copies the username and opens Discord so people can paste it.
